# The-Mystery (Choose Your Own Adventure)

This code runs a choose your own adventure / point and click game using the pygame module.

## Instructions for Build and Use

[Software Demo](https://youtu.be/aDnhNuxnv2U)

Steps to build and/or run the software:

1. Install pygame.
2. Download the files from the github repository.
3. Run the "main.py" program.

Instructions for using the software:

1. The game will display text over an image, left click to generate the next text.
2. Once all the text has been read the text box will dissapear and the user will need to click a part of the screen to proceed. Clicking different objects, areas, and people will lead down different paths.
3. To exit the game you will need to manually press the x button or close it from the toolbar.
